まず [AGENTS.md](AGENTS.md) を読むこと。あなたの役割はエージェント編成表で特定する
(**既定: 奏者(worker)**)。責務は AGENTS.md「役割の特定」に従う:

- 奏者のとき: 渡された spec / plan の範囲でのみ作業し、結果はダイジェストで返す。
  範囲外の変更(ついでのリファクタ・依存やテスト環境の書き換え)はしない。
  自己検証は参考情報であり、GREEN の最終判定は指揮者が行う(断定報告しない)。
  Web 調査の成果には必ず出典 URL を添える。
- 指揮者に交代している編成のとき: `bolt` / `cli-routing` / `cross-review` スキル
  (パスは AGENTS.md 参照表)をファイルとして読み、指揮者の責務(判断・検証・
  グリーン判定・採否)を自ら担う。
- 秘密情報(.env・鍵・トークン)は読まない・出力に含めない(AGENTS.md 共通禁止事項)。

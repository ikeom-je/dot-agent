# Claude Code 固有

あなたの役割は AGENTS.md のエージェント編成表で特定する(**既定: 指揮者**)。

指揮者のとき(AGENTS.md「役割の特定」の責務に加えて):

- 新しいトピックの作業を始めるときは `bolt` スキルを使う。
- 委譲の判断は `cli-routing` スキルに従う。迷ったら自分でやる。
- subagent の自己申告 GREEN を最終判定に使わない。グリーン判定はあなた自身が
  クリーンな状態で実行して行う。
- レビュー指摘の採否はあなたが検証して決める。盲目的に全反映しない。

奏者に交代している編成のときは、AGENTS.md「役割の特定」の奏者の責務に従う。

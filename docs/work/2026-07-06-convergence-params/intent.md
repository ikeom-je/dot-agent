# intent: convergence-params

- 解きたい課題: 収束ループ(検証・相互レビュー)が実装フェーズにしか回数制限付きで
  定義されておらず、上流成果物(intent/PRFAQ/UX調査/spec/plan)には掛かっていない。
  また回数が複数docにハードコードされ、プロジェクトごとに調整できない。
- この bolt でやること:
  1. 全フェーズ共通の「作成→検証→相互レビュー→収束」ミニループを lifecycle/review に定義
  2. 収束パラメータを AGENTS.md の一箇所に表として集約し、各docはパラメータ名で参照
  3. スキル(bolt/prfaq/ux-research)にレビューゲートを追加
- やらないこと: パラメータの機械的な読み込み(設定ファイル化)。docベースで十分
- 完了の定義: 全フェーズにレビューゲートがあり、回数の数値が AGENTS.md 以外に
  ハードコードされていない。リンク検証グリーン
